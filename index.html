Projects
Blackjack Markov Decision Process Winter 2019

Models the Blackjack card game as a Markov Decision Process(MDP) in order to calcuate optimal move tables without simulation though value iteration. It also allows the user to input a card counting strategy and generate optimal move tables at each hand.
Adversarial Games and Multi-Agent Search CS 188, Summer 2020

We design agents for classic Pacman(including the ghosts), and implement minimax(w/ alpha-beta pruning) and expectimax search with custom evaluation functions.

Machine Learning: Digit Classification, Language ID CS 188, Summer 2020

We build neural networks to classify digits, identify languages, and approximate non-linear functions.
Ghostbusters CS 188, Summer 2020

We design Pacman agents that use sensors to locate and eat invisible ghosts, with Exact Inference, Approximate Inference w/ Sampling, and Particle Filter methods.

Transversal Properties of Families of Convex Sets Renyí Institute, Fall 2020

Group research in convex geometry under the supervision of Gergely Ambrus. We study relaxations of Helly’s theorem in order to characterize transversal properties of families of convex sets.

Geodesic Convex Optimization CS 270, Spring 2021

Defines Geodesic Convexity in order to perform convex optimization techniques on Riemannian manifolds with curvature constraints. Includes geodesic convex functions, operator scaling, optimistic likelihood calculation, sampling via geodesic random walks, and optimization via geodesic gradient descent and simulated annealing. Worked with Jacob Krantz and Rohith Sajith.
Using the Gaussianized Coding Rate for Generative Modeling EECS 208, Fall 2021

Theoretical and Experimental Analysis of the Gaussianed Coding Rate for distribution learning tasks.
